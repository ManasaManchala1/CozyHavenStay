# Cozy Haven – Hotel Booking System

A full-stack hotel reservation web application built as a collaborative 2-person project.

## Overview

Cozy Haven simulates a real-world hotel booking platform — users can browse available rooms, make reservations, and manage their bookings, with secure authentication protecting user and booking data.

## Tech Stack

- **Frontend:** React.js
- **Backend:** C# / .NET
- **Database:** SQL Server
- **Authentication:** JWT (JSON Web Tokens)

## My Role

I owned the **backend**: designed and built the C#/.NET API layer, the SQL Server database schema, and implemented JWT-based authentication and authorization to secure user and booking data. My teammate built the React.js frontend.

## Features

- User registration and login secured with JWT-based authentication
- Room browsing and hotel booking workflow
- Booking and user management features simulating real-world reservation flows
- Role-based access control for secure data handling

## Getting Started

This project has two parts:
- **Backend** — a C#/.NET API with a SQL Server database
- **Frontend** — a React.js client

To run it locally, you'll need the .NET SDK and Node.js installed, with the backend's database connection configured for your local SQL Server instance.
